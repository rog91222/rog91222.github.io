写在前面

就像这个分类名「自我厌恶与自我救赎中的思考」
成长本就是撕开认知假象的过程，
每经过一件事儿，就会发现一些问题，积极改正就好。

关于学业

上了大学之后明显感觉到不同层次人的水平差距，也只有相同水平层次的人才能组成圈子。
人的层次是和木桶短板原理相似：「学历、能力、资源、实力、语言、性格…最短的那块木板，决定了你的最高层次」
抛去性格等几个无法量化的因素，学历无疑是最短的那块了。
从一月到三月都在准备「专转本考试」，从填报志愿时估计的200分谢天谢地，到考前估分250分，再到最终297分的成绩，
结果还算满意，现在也是个本科生了。

虽然在2014-2016这三年里工作和学习方面还算努力，但是在以前浑浑噩噩的生活方式给性格造成的影响下，
始终没有能够完整安静的学完一项知识，就是常说的「坐不住」;
备考的三个月里，尝试着把自己封闭起来，说真的,手机关机的感觉真好。
学习本就是孤独的事情，要想找到一个能够独处、封闭、舒适的学习环境是件听不容易的事儿，
有了环境,又伴随着焦虑、孤独、崩溃等情绪，在这里要感谢我坤@Quentin_Zhu，复习中一路搭档，才有了我数学接近满分的成绩。
所谓搭档就是「我的矛，也是我的盾」，既需要在遇到问题时提出可行的建议，也需要相互倾泻负能量。
(请远离单方面倾泻负能量的朋友，这是耍流氓)

大专三年的日子里虽然成绩说得过去(绩点4.0+)，扪心自问作为学生是不合格的，很少有认真听课的时候。
带着对前三年的愧疚感,本科第一学期过得格外…辛苦
五科考试课一学期下来堆了200多页的笔记下来，虽然没有挂科的担心，幸运的话还能蹭到奖学金。

但是如果用一句话总结9-12月，大概就是在「浪费时间」。
半年里的大部分时间都在忙着上课的事情，Vue还没学完，JavaScript已经忘得差不多了，在新的一年里平衡好上课与专业的时间问题。

英语……不提了不提了，英语再见，英语再见。

「2017」收获

  1. 深刻体会到，试卷分析/历年情报的重要性
  2. 找到了适合自己的学习方法
  3. 初尝封闭起来学习的甜头，但有时拗不过自己
  4. 对待学习的态度端正很多

「2018」期望

  1. 平衡好上课与专业的时间分配
  2. 提高学习时的专注度
  3. 课程60分万岁，多了不要了

关于运动

从小对运动一窍不通而一度被教练嫌弃。健身的时候别人学一次就能做标准的动作，我需要学一个星期。
朋友吐槽我：人有三商，你的智商和情商勉强说得过去，至于动商…要不重新点一下天赋技能吧。
在跑步和跳绳中挣扎了半年，也没有坚持下来。
幸运的是这段时间找到了本命运动「羽毛球」，来到新学校之后认识了很多一起切磋/研究羽球的朋友，还是蛮好的;
希望2018年能够继续练好，真正入了羽球的门。

「2017」收获

  1. 明白了如何提升羽球技术
  2. 收获一群野生球友

「2018」期望

  1. 热情不减，真正入门
  2. 在体能训练上多下功夫
  3. 技术能够有提升
  4. 保护好关节

关于笔记

用OneNote的第三个年头,
「514篇」
每多记一篇笔记，对「遗忘」就多忌惮三分。
六月的时候把笔记全部清理了一遍，将摘录的或是整篇复制的(甚至有些都没有吸收就贴进来的)剔除出去，重新来过。
删删改改后留了300有余，年末的时候又增加到了500篇，质量提升了很多。
毕竟只有消化理解之后产出的内容才是自己的。

「2017」收获

  1. 找到了适合的排版，干净大方
  2. 将进度安排用OneNote记录，对进度有把握
  3. 来不及弄明白的遗留问题汇总记录
  4. 能够非常熟练使用公式

「2018」期望

  1. 更多的关注点放在知识本身而不是做笔记上
  2. 减少摘抄和整段复制
  3. 减少遗留的问题，优先解决
  4. 减少做笔记过程中耗费过多精力

关于博客

五月在博客园开了自己的博客，十月迁移到了自己的服务器上；
刚开始选择博客园是因为OneNote可以直接导入，
但笔记于别人来说是毫无用处的，大概需要经历这几个阶段：
会记笔记 → 会写注释 → 会写教程 → 会写文章
逐步从「抄别人的」到「讲自己的」，从「给别人看」到「让别人懂」的过程。
需要长时间的沉淀，慢慢来吧。

迁移到自己的服务器也是偶然，
一是刚接触到MarkDown，二是OneNote导入博客园后格式很难修改。
因为服务器有一台在手里闲着，就没用GitHub空间，索性就自己搭起来了。
搭建过程见 传送门

「2017」收获

  1. 博客园「73篇」 Hexo「16篇」
  2. 从刚开始的不知道如何表达，慢慢能够把所想的内容记录下来
  3. 借着搭建博客将Docker部署摸熟悉了
  4. 将所学的内容细化，讲给别人听的过程也验证了自己是否明白

「2018」期望

  1. 写作风格再成熟些，现在偏轻佻幼稚
  2. 博客SEO优化
  3. 博客前端样式优化
  4. 扩展到MarkDown可以统一的其他平台(简书)
  5. 恢复微信公众号的运营

四大类支出情况

类名 2017金额 占比 2016金额 占比
生活 ¥8,727.60 27.15% ¥8,644.90 35.25%

从数额来看，今年在生活上的基础开支与去年基本相同，其他三类都增长了不少；

就明细来看，多的部分主要是用在买书与英语四级(这样都拯救不了的英语水平)，学费也比之前多了一些。

月份支出情况
从月份上的情况来看，九月开销最大，来到新学校购置生活用品。
一月/三月/八月/十月…钱都吃肚子里去了(捂脸)

租车行业调研报告

2018-05-08 | 「业务」范式


声明：涉及本公司部分信息已经过删改，其他数据均来源于互联网，转载请注明出处。

一、行业概况

1.1 商业模式

全国性租车平台
神州、一嗨、悟空等租车平台均使用B2C模式，B2C模式的典型特征是：租赁企业自营车辆租赁平台；自主采购车辆资源；自建线下门店；车辆、产品、服务、流程标准化，具备品牌、规模效。
部分企业采用B2C+平台模式，接受传统租赁公司的加盟，获取车辆资源。典型的有一嗨租车、易租车、大方租车。
全国性租车平台的业务主要以短租、长租和分时租赁业务为主，在此基础上各平台提供“闪租”（一嗨租车）等特色业务。

P2P平台
P2P平台是连接出租用户与承租用户中间平台，典型企业有PP租车、凹凸租车、共享有车生活平台。将家庭闲置车辆加盟平台，供给用户使用，车主能够随时监控车辆位置、行驶情况。

中小公司/车队
地方性中小型租车公司或车队主要承接旅游、商务长租和婚庆租车业务。

1.2 业务模式

租车平台自购车辆并办理营运资质，同时负责车辆的保养、维修与保险赔偿。租车人发生违章时通过平台协调进行违章处理，扣除分数以及缴纳罚款。详细内容在经营细节讨论。车辆租赁模式基本相同，在租车时间长短上主要分为长期、短期和分时租赁,使用场景如下所示：

长期租赁 短期租赁 分时租赁
使用对象 以企业客户为主，尤其外企高管、领导，少数个人用户 个人用户为主，满足个人短期用车需求 目前以一二线城市年轻白领群体为主
使用场景 企业日常车队用车；外企、国企高管日常用车；酒店、旅行社等日常用车 城市周边自驾出游；旅游目的地用车，包括出境游目的地的国际租车；商务接待、商务差旅等 临时、短途商务出行；日常上下班、周末市内出行

1.3 收入模式

目前，中国汽车租赁市场主要营利模式较为清晰,收入来源于三部分：

  1. 车辆租赁收入
  2. 二手车残值处理
  3. 保险业务

其中，车辆租赁收入和二手车出售收入是主要收入来源，收入能占企业总收入70%。另外，车辆采购成本和车辆运营成本是最大两块固定支出项目，尤其车辆采购成本与企业采购规模直接相关，直接影响企业的盈利水平。所以，形成规模化经营，加强与上游厂商合作，降低采购成本，最大力利用车辆价值，合理规划车辆使用周期，提升二手车残值处理能力是企业盈利的关键，这也是大型租赁企业优势所在。

除此之外，保险业务是租车行业公认的重要收入来源之一。
租车时的费用分成这几块：

  1. 车的租金（必购）
  2. 基本保险加手续费（必购）
  3. 增值服务（选购）：包含导航、不计免赔、儿童座椅、送车服务等

在不计免赔服务上通常租车平台是有盈利的。例如租车发生追尾事故，定损后需赔付2650元，按其与神州租车签订的合同因未购买50元/天的不计免赔服务，租车人需承担1500元的费用，而保险公司和神州租车只需共同承担1150元。保险公司表示会根据与神州租车公司间的保险合约规定来进行赔偿，会全额赔付2650元。而投保1500元“不计免赔”，实际成本仅2元/天。
在不计免赔服务的模式下，当租车人发生小的擦碰事故时，租车平台就能够利用协议上的漏洞赚取相当可观的费用。

1.4 市场分布

据统计，国内租赁汽车车队总规模约为80万台左右，前四大汽车租赁企业运营车队总规模约23万辆，占比近30%。尚有70%的运营车辆分布在全国各地，包括地方性国有租车公司以及车队体量在500台以下的民营租车公司，各式各样的租车品牌超过5,000个。

从2013年开始，神州租车、一嗨租车就已经对一线主力城市之外进行布局。但根据最近调查的情况来看，二三线城市租车发展仍然处于迟滞的状态。在和徐州地区易到租车的业务讨论挂靠的可行性时提到徐州市场购车需求远大于租车。无论是私家使用还是专车司机在成本相差不大的情况下都更倾向于购车。
徐州市场上的网约车来源有两种：自购和以租代购，除了自己购买外以租代购也是主要的形式，以租赁的形式按期支付租金（月供3000左右），租赁一定期限（一般是3年）后车辆所有权归专车司机所有。除了这两种车辆来源外，向网约车司机提供纯租赁车辆这种形式在徐州地区尚没有租车公司使用。

1.5 加盟合作

全国性租车平台中加盟合作情况分为三种：自营、接受公司加盟、接受司机加盟。加盟合作的模式汇总情况如下：

  1. 完全公司自营：华晨赶脚租车、首汽租车；
  2. 接受公司加盟：悟空租车、一嗨租车、大方租车、易到租车、易租车、滴滴租车；
  3. 接受司机加盟：神州租车、滴滴租车。

经过调研发现租车平台不接受车辆挂靠的模式，车辆均由平台出资购买。
二手车的处理是租车平台的重要收入来源之一，平台自购车辆比租赁车辆的成本要低很多。大多数租车公司与车厂签订大批量购车订单时价格约为市场价的60~70%，而租车平台的车辆下线期限为30~36个月。按照行业平均短租出租率60%，租日平均里程200~250公里来计算。365 day*3 year*60%*225 km=147,825 km≈150,000 km
从购入新车到三年后车辆下线，平均里程数在15万公里左右。
在二手车之家平台上可以折算二手车的价格，以华颂7（2015款 2.0T）三年车龄、行车15w公里且车况偏差的情况下得到以下数据：

在表中可以看到，租车平台在7折购入且三年后顺利售出的情况下，每个月的车辆成本低至1500元，而且完全持有车辆。在这种情况下租车平台接受车辆挂靠增加了用车成本。
以神州租车的商业模式为例 ，主要业务收入来自于租车服务、二手车残值处理与保险业务。而在神州的全产业链闭环中，神州买买车提供了下线二手车的出口，自购车辆能够顺利处置。

经过上述分析，车辆挂靠到租车平台上与其商业模式产生冲突，影响了平台收益，所以互联网租车平台的商业模式中不接受车辆挂靠合作。

二、经营细节

2.1 公司资质

成立租车公司需要具备以下技术经济条件：

  1. 配备汽车不少于20辆，且汽车车辆价值不少于200万元。租赁汽车应是新车或达到一级技术等级的在用车，并具有齐全有效的车辆行驶证件。
  2. 须有不少于汽车车辆价值5%的流动资金。
  3. 有固定的经营和办公场所，停车场面积不少于正常保有租赁汽车投影面积的1.5倍。
  4. 有必要的经营机构和相应的管理人员，在经营管理、车辆技术、财务会计等岗位上应分别有一名具有初级及其以上职称的专业技术人员。
  5. 具有法人资格。
  6. 2人或2人以上有限公司注册资金要3万以上。
  7. 有限公司注册资金要10万以上可先去工商行政管理部门进行公司名称预登，如果经审核名称无重复并适用，则发予“名称预先登记核准通知书”，再凭此证到市交通部门办理道路运输经营许可证。拿到许可证后，可以带上身份证、房产证复印件、验资报告到工商部门办理最后的准予手续。

根据以上条件计算，成立租车公司所需要的基础投资除了车辆成本、加盟费用、门店租金外，还包括：

  1. 车辆场地成本：20辆华颂7所需停车场地面积最低为 5.104m*1.909m*1.5*20=292.306080m^2≈300m^2。
  2. 流动资金：20辆低配华颂7所需的流动资金最低为 20*23万元*5%=23万元
  3. 运营成本：以神州租车2014Q1短租情况计算，在运短租车辆3.5万台、出租率60.9%，成交46.3万宗，得到每宗租赁为4.14天，收到租金1163元。每完成一宗短租交易，神州租车运营成本高达420元。在车辆少的情况下，保险租金、保养维修费用都会成倍增加。

2.2 申领手续

申请经营汽车租赁业务应按下列程序办理：

  1. 持街道、乡级人民政府或主管部门证明和企业组织章程、经营可行性分析报告、资信证明等有关材料向当地县级道路运政管理机构提出开业申请，由县级道路 运政管理机构审核后，报地市级道路运政管理机构审批。
  2. 地市级道路运政管理机构审查批准后，核发《道路运输经营许可证》。
  3. 凭《道路运输经营许可证》到工商行政管理、税务等部门办理有关手续。
  4. 按批准的数量购置车辆，办理租赁汽车入户并按有关规定办理车辆第三者责任险，到道路运政管理机构领取营业性《道路运输证》。

2.3 车辆保险

神州、一嗨、悟空等租车公司对外宣传的是： “车辆损失险、20万元第三者责任险、5万元车上人员责任险”等，甚至还有公司称能提供全车盗抢险、玻璃单独爆裂险、自燃损失险等。实际上大多仅购买了交强险和5万元商业险，甚至有的只购买了交强险，仅有极少车辆配备了10万元以上的商业三者险。
车辆发生实际损失时，租车人需要承担1500元及以内的车损险责任，保险公司和租车公司承担1500元以上的部分。购买5万的第三者商业保险加上交强险，距离20万元的保额相差两三万。通常租车公司的做法是自掏腰包支付差额，而避免每台车增加更多的保额费用。
在和保险公司相关人员沟通之后了解到，大规模的租车公司（如神州租车）是在保险公司购买交强险后，交强险之外的赔偿金额由公司自己赔付。
大部分保险公司不给租车公司投保，原因是出险频度太高。在愿意承保的保险公司中，仅展示某保险给予的最低价格，仅包含必备险种（交强险、车损险、三者和不计免赔）的情况报价如下：

2.4 违章处理

租车人在租车时需要冻结违章抵押金，车辆违章时需要由租车人自己来处理。处理违章时需要身份证、驾驶证和行驶证原件，在车辆规模小时可以派专人负责违章消除，但当公司规模大时，部分公司的处理办法是与车管所沟通关系（重点），只需要传给其三证复印件来消除违章。

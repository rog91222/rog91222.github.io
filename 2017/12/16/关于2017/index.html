2017-12-16 | 自我厌恶与自我救赎中的思考


写在前面

就像这个分类名「自我厌恶与自我救赎中的思考」
成长本就是撕开认知假象的过程,
每经过一件事儿,就会发现一些问题,积极改正就好

关于笔记

用OneNote的第三个年头,
「514篇」
每多记一篇笔记,对「遗忘」就多忌惮三分,
六月的时候把笔记全部清理了一遍,
将摘录的或是整篇复制的(甚至有些都没有吸收就贴进来的)都剔除出去,重新来过,
删删改改后留了300有余,年末的时候又增加到了500篇,质量提升了很多
毕竟只有消化理解之后产出的内容才是自己的。

「2017」收获

  1. 找到了适合的排版,干净大方
  2. 将进度安排用OneNote记录,对进度有把握
  3. 来不及弄明白的遗留问题汇总记录,有迹可循
  4. 能够非常熟练使用公式

「2018」改进

  1. 更多的关注点放在知识本身而不是做笔记上
  2. 减少摘抄和整段复制
  3. 减少遗留的问题,优先解决
  4. 减少做笔记过程中耗费过多精力

关于博客

五月在博客园开了自己的博客,十月迁移到了自己的服务器上
博客园「73篇」
Hexo「16篇」

刚开始选择博客园是因为OneNote可以直接导入,
但笔记于别人来说是毫无用处的,大概需要经历这几个阶段:
会记笔记 → 会写注释 → 会写教程 → 会写文章
逐步从「抄别人的」到「讲自己的」,从「给别人看」到「让别人懂」的过程
需要长时间的沉淀,慢慢来吧。

迁移到自己的服务器也是偶然,
一是刚接触到MarkDown,
二是OneNote导入博客园后格式很难修改,
因为服务器有一台在手里闲着,就没用GitHub空间,索性就自己搭起来了
搭建过程见 传送门

「2017」收获

  1. 从刚开始的不知道如何表达,慢慢能够把所想的内容记录下来
  2. 借着搭建博客将Docker部署摸熟悉了
  3. 将所学的内容细化,讲给别人听的过程也验证了自己是否明白

「2018」改进

  1. 写作风格再成熟些,现在偏轻佻幼稚
  2. 博客SEO优化
  3. 博客前端样式优化
  4. 扩展到MarkDown可以统一的其他平台(简书)
  5. 恢复微信公众号的运营

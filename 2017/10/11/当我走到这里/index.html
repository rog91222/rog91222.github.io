当我走到这里

Veröffentlicht am 2017-10-11
联系博主

好久不见,是我.
码字儿的人总是乐于折(zuo)腾(si)生命,无论是程序员还是自媒体,大都如此.
从博客园搬到阿里云主机上,图个静下心码字儿的仪式感.
回头看看来时的路,陋习改之,长处共勉.
可我不认识你呀… ฅ’ω’ฅ 那我们现在不就认识了.

我是谁

# 本科在读,要做一个认真而且愚蠢的人
# 什么都会一点的半吊子
# 主业修电脑的,副业帮前端打杂,定位甲方三陪业务岗
# …
# 毕业于中央戏精学院,擅长毒奶和FLAG
# 沉迷二次元/暴雪/日本文学/讨论人性的电影
# 常伴抹茶/提拉米苏/普洱/黄酒/冰咖
# 摩羯座,喜欢随身带东西,通勤包重重重重重重重
# 强迫症晚期抢救中
# 谁心里没住个小公举呢…(‘-ωก̀ )
# …

技能树

从2014年高考之后,受家人的影响,开了自己的小店卖电脑,专科时读的是网络专业
少不更事,在不同的方向中跳来跳去,最终得到了一个乱糟糟的技能栈
别人的技能树愈发强壮,我立了很多树墩,宛如戏台上老将军背后的的FLAG
刚进入大学的少年们呀,不要再问哪个方向更赚钱更好找工作了
你的技能学的有多深入,便会获得一个好价钱
你需要做的就是
>> 找到你最喜欢做的那件事
>> 一直一直一直走下去
时间不会说谎,也不会辜负任何人
你付出多少了时间,便会收获多少成果
加油吧,少年,不要在方向之间踌躇选择了,去放手学吧

如果志趣相投,希望你能够联系我,毕竟有趣的灵魂难找呢
